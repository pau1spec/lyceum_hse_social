# Lyceum Social

## Возможности

- Регистрация пользователей
- Авторизация (JWT)
- Автоматический вход
- Создание групп
- Просмотр списка групп
- Поиск групп
- Поиск пользователей
- Создание постов
- Просмотр постов внутри группы
- Лайки постов
- Удаление постов
- Комментарии к постам
- Удаление комментариев

## Использовано 

- Frontend: React + TypeScript (Vite)
- Backend: NestJS
- База данных: PostgreSQL (Docker)
- ORM: Prisma
